## Semantics
Semantics is the meaning of the code once it is written correctly. It explains how the program behaves.

**Example:**
```python
for i in range(5):
    print(i)
```

**Semantics:**
- `range(5)` â†’ `[0, 1, 2, 3, 4]`
- Variable `i` takes each value
- Prints each number

**Output:**
```
0
1
2
3
4
```

ðŸ‘‰ Syntax may be correct but semantics wrong (e.g., `range("five")`) â†’ runtime error.
